TrendyTools cPanel Site Builder suite for Web Host.

cPanel has grown to become the most popular, and first, commercially-available control panel software in the world. cPanel is a Linux based web hosting control panel. cPanel offers web hosting software that automates the intricate workings of web hosting servers. cPanel products are used on tens of thousands of servers worldwide to equip server administrators with the tools they need to provide top notch hosting to their customers.

TrendyTools is proud to partner with all the cPanel hosting companies. We have over 400 web hosting companies that use the HTML5 cPanel site builder suite. TrendyTools is currently available in over 10 languages such as Spanish, Portuguese, French, Turkish, Japanese and more.

With the addition of new specialized site builders for Salon, Church, Restaurant, Band, Attorney, Pizza, Hotel, Photography and Real Estate Web hosts can now be sure of providing highly targeted site building capabilities to more of their customers.

TrendyTools Website Builder Suite for cPanel Control Panel is the most popular cPanel plugin in cPanel application catalog.

Integrating Trendy Tools into cPanel

Prerequisites:


Installation Procedure:

Using SSH, log into your server as root. If you do not have root access, have your system administrator perform these steps:

$ root@my-cpanel-server.com [~]#

Download the installer from trendyflashdownload.com
root@my-cpanel-server.com [~]# wget "http://www.trendyflashdownload.com/installer/cp/sitebuilders/trendytools.sh"

Set execute permission for trendytools.sh
root@my-cpanel-server.com [~]# chmod 755 trendytools.sh

Run the shellscript trendytools.sh
root@my-cpanel-server.com [~]# ./trendytools.sh

Now your users can see the "Trendy Tools Icon" under the Software/Services Group in their cPanel.


Edit License key:

Change the directory to /usr/local/cpanel/base/frontend/x3/trendytools
root@my-cpanel-server.com [~]# cd /usr/local/cpanel/base/frontend/x3/trendytools

Using Vi Editor you can edit your license key in editkey.php
root@my-cpanel-server.com [~]# vi editkey.php
