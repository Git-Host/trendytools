ZPanel control panel

ZPanel HTML5 WebSite Builder – TrendyTools

ZPanel is a free to download and use Web hosting control panel written to work effortlessly with Microsoft Windows and POSIX (Linux, UNIX and MacOSX) based servers or computers. This solution can turn a home or professional server into a fully fledged, easy to use and manage web hosting server.

The software utilizes other free or open-source software to provide a ‘license’ free, cost effective solution. ZPanel flawlessly integrates with MySQL, hMailServer (on Windows), Posfix (on Linux etc.) and Filezilla (on Windows) or ProFTPd (on Linux etc.). ZPanel already powers many web hosting companies, organizations and home users that want to host websites or provide web hosting services the software makes it really simple to offer a simple and easy way to automate hosting whether it’s email, web or database.

TrendyTools has decided to facilitate every ZPanel web host on Linux with state of the art website builder. zPanel Website Builder offers web hosting clients with a Html 5 web presence. All the websites creates using ZPanel site builder are responsive in tablets and mobile devices. TrendyTools Site Builder suite for ZPanel will include website builders for Business, Band, Salon, Church, Pizzza Shops, Doctors, Real estate agents and more. All the site builders in the ZPanel suite will be available in more than 10 languages.

Integrating Trendy Tools into ZPanel

Prerequisites:


Installation Procedure:

Using SSH, log into your server as root. If you do not have root access, have your system administrator perform these steps:

Install Trendy Tools Module by adding to Zppy repository
root@my-zpanel-server.com [~]# zppy repo add partners.trendytools.com/zpanel

root@my-zpanel-server.com [~]# zppy update

root@my-zpanel-server.com [~]# zppy install trendy_tools

root@my-zpanel-server.com [~]# zppy install trendy_tools

Set write permissions for editkey.php file
root@my-zpanel-server.com [~]# chmod 777 /etc/zpanel/panel/modules/trendy_tools/code/editkey.php

Edit License Key in editkey.php at the location "/etc/zpanel/panel/modules/trendy_tools/code/editkey.php"

Enable Trendy Tools Module For Administrators,Resellers,Users in "Module Admin" Link under Server Admin.

"Trendy Tools" icon now appears in Advanced section
