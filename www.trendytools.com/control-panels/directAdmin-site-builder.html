DirectAdmin control panel

TrendyTools DirectAdmin website Builder suite for Web Host.

DirectAdmin is the easiest to use Linux control panel. DirectAdmin has been programmed to be the fast. Succeeding in the web hosting business requires flexibility which DirectAdmin offers.

With unrelenting requests from the DirectAdmin web hosts TrendyTools has launched the Site Builder plugin for DirectAdmin Control Panel. The website builder for DirectAdmin will empower the webs hosts to offer a highly targeted site building experience to their clients.

TrendyTools Site Builder suite for DirectAdmin will include website builders for Business, Band, Salon, Church, Pizzza Shops, Doctors, Real estate agents and more. All the site builders in the DirectAdmin suite will be available in more than 10 languages.

Integrating Trendy Tools into directAdmin

Prerequisites:


Installation Procedure:

Using SSH, log into your server as root. If you do not have root access, have your system administrator perform these steps:

$ root@my-directadmin-server.com [~]#

Download the installer from trendyflashdownload.com
root@my-directadmin-server.com [~]# wget "http://www.trendyflashdownload.com/installer/directadmin/trendytools.sh"

Set execute permission for trendytools.sh
root@my-directadmin-server.com [~]# chmod 755 trendytools.sh

Run the shellscript trendytools.sh
root@my-directadmin-server.com [~]# ./trendytools.sh
