interWORX control panel

Interworx Website Builder by TrendyTools

The InterWorx Hosting Control Panel is a web hosting and linux server management system that provides tools for server admins to command their servers and for end users to oversee the operations of their website.

TrendyTools is very happy to be associated with Interworx. The Interworx website builder will have all the features of the latest technology TrendyTools offers its web hosting clients. TrendyTools is currently available in over 10 languages such as Spanish, Portuguese, French, Turkish, Japanese and more. TrendyTools website builder for Interworx will the first Html5 website builder Interworx clients will get.

The installation process is very simple and clients can apply the license directly from the Interworx control panel. TrendyTools Site Builder suite for Interworx will include website builders for Business, Band, Salon, Church, Pizzza Shops, Doctors, Real estate agents and more. All the site builders in the DirectAdmin suite will be available in more than 10 languages.

The panel has two operating modes: Nodeworx and Siteworx. NodeWorx is the system administrator mode for managing server-wide behavior and SiteWorx accounts. SiteWorx is the website owner mode and is used primarily by hosting end-users to manage their web hosting accounts and associated features.

Integrating Trendy Tools into interWORX

Prerequisites:


Installation Procedure:

Using SSH, log into your server as root. If you do not have root access, have your system administrator perform these steps:

$ root@my-directadmin-server.com [~]#

Download the installer from trendyflashdownload.com
root@my-interworx-server.com [~]# wget "http://www.trendyflashdownload.com/installer/interworx/trendytools.sh"

Set execute permission for trendytools.sh
root@my-interworx-server.com [~]# chmod 755 trendytools.sh

Run the shellscript trendytools.sh
root@my-interworx-server.com [~]# ./trendytools.sh
